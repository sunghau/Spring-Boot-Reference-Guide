###22. SpringApplication
SpringApplication類提供了一種從main()方法啟動Spring應用的便捷方式。在很多情況下，你隻需委托給SpringApplication.run這個靜態方法：
```java
public static void main(String[] args){
    SpringApplication.run(MySpringConfiguration.class, args);
}
```
